OUR GOAL


As we know that Breast cancer can occur in women and rarely in men. So we just decided to build a ML model that can predict this type of cancer with tumour i.e., malignant tumour or benign tumour. Our ML model will predict the output by analyzing and applying machine learning algorithms with correct input features.


  • Model can predict the result of tumour by adding 30 features as an input with 97% accuracy.
  • The precision for benign tumour for this model is 94% and 98% for malignant tumour.
  • As it is based on ML model so it can be accessed easily at low cost.
  • Breast cancer risk prediction models used in clinical practice have low discriminatory accuracy (0.53 - 0.64)
